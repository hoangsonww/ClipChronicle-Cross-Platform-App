# Install Landing Site

1. **Clone repo**

   ```bash
   git clone https://github.com/your-org/clipchronicle.git
   cd clipchronicle/clipchronicle-landing
   ```

2. **Install dependencies**

   ```bash
   pnpm install
   ```

3. **Run dev server**

   ```bash
   pnpm dev
   ```

4. **Build & export**

   ```bash
   pnpm build
   pnpm export
   ```

5. **Deploy**
   Sync `out/` to your S3 bucket or static host.
