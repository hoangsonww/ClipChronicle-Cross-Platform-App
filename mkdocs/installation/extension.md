# Install Browser Extension

1. **Navigate**

   ```bash
   cd clipchronicle/extension
   ```

2. **Install & build**

   ```bash
   pnpm install
   pnpm build
   ```

3. **Load in Chrome**

- Go to `chrome://extensions`
- Enable **Developer mode**
- Click **Load unpacked** and select `dist/`
